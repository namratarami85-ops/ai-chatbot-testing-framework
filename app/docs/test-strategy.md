# AI Chatbot Testing Strategy

## Objective
This project tests AI chatbot quality across key QA risk areas.

## Test Areas
- Response accuracy
- Prompt injection resistance
- Hallucination handling
- Reliability and consistency

## Current Coverage
- Basic response validation
- Prompt injection test case
- Dataset-based prompt storage

## Future Enhancements
- Real API integration
- Automated reporting
