# AI Chatbot Testing Strategy

## Objective
This project tests AI chatbot quality across several key areas.

## Test Categories

### Accuracy
Validate correct responses to known questions.

### Prompt Injection
Ensure the chatbot does not expose system instructions.

### Hallucination Prevention
Ensure the chatbot does not fabricate answers when it does not know the answer.

## Current Scope
- Mock chatbot testing
- Dataset-driven testing
- Basic QA automation structure

## Future Enhancements
- Connect to real AI APIs
- Automated test reports
- Continuous Integration testing
